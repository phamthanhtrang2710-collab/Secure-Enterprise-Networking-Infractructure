# Secure-Enterprise-Networking-Infractructure
This project shows the design, deployment, security hardening, monitoring, automation, and troubleshooting of a secure enterprise network infrastructure.

The environment simulates a small-to-medium business network with multiple departments, redundant gateways, dynamic routing, secure access control, server infrastructure, monitoring, and basic network automation.

## Business Scenario
A fictional company named <em>"Verra Technology Inc."</em> is a growing technology company with a head office, branch office, internal servers, remote workers, and multiple departments.

The company requires a secure, scalable, and highly available network infrastructure to support daily business operations, internal communication, internet access, centralized services, and future growth.

## What does this project aim to archive?

* Design an enterprise network architecture

* Segment business departments using VLANs

* Deploy OSPF and BGP routing

* Implement HSRP for gateway redundancy

* Secure inter-VLAN communication with ACLs

* Deploy centralized network services

* Monitor network health and performance

* Automate configuration backup with Python

* Validate the infrastructure through testing and troubleshooting
